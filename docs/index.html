創立人 吳承鴻 先生，早期專業於3C電子消費暨電腦周邊產品之模具開發及製造，具有20幾年以上之模具技術與塑膠射出品製造經驗。
為能服務更寬廣的客戶組群，於2004年3月11日成立羽泰精密科技有限公司，結合原有專業及生產技術、開發新型生產治具降低產品生產成本為前提，專營產品組裝及代工服務。
並於2005年5月16日通過 ISO 9001:2008 認證，導入認證流程及制度，嚴格控管品質政策，由於率先掌握關鍵性的技術而在業界居於領先之優勢，屢獲知名大廠不斷邀約及高度的肯定。
